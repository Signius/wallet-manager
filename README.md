# Deploying to Netlify

This app is a Next.js project. If you deploy on Netlify, you **must** use the Netlify Next.js plugin so `/_next/static/*` assets and Next routing work correctly.

- **Config**: `netlify.toml` (included in this repo)
- **Build command**: `npm run build`
- **Publish directory**: `.next`
